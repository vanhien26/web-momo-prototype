# MoMo Web Platform Prototype Lab

Static prototype showcase for Web Platform product concepts:

- Money Lab
- Merchant Discovery
- World Cup 2026 Predictor

## Run locally

```bash
python3 -m http.server 4173
```

Open `http://127.0.0.1:4173`.

## Scope

This repository is a research and product-spec prototype. Calculations, merchant data, offers, rankings, and probabilities are illustrative and must not be treated as production data or financial advice.
